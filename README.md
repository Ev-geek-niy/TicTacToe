# Tic-Tac-Toe

Это небольшая консольная игра в крестики-нолики.

## В игре реализована вся необходимая логика для полноценной игры:
* Проверка на победу
* Проверка на то, что клетка уже закрашена игроком
* Проверка на то, что игрок не может выбрать номер клетки за пределами возможного

## Стек

Ничего, только стандартные плюсы (а что еще могло быть)

## Установка
* Установить
* Скомпилировать
* Играть
